What are Claude Skills?

Skills are specialized AI agents built on the Anthropic Claude Agent SDK. Each skill is a self-contained SKILL.md file that gives Claude a specific identity, toolset, and knowledge base — turning it from a general assistant into a domain expert. Skills are invoked via slash commands (e.g. /aio-mastermind) or natural language triggers and can connect to live data sources like Slack, Smartsheets, and Google Drive.

How It Works

Architecture

Each skill is a SKILL.md file with a YAML frontmatter block defining its name, description, and trigger conditions. When invoked, Claude reads the skill file and adopts the defined persona and toolset. Skills access live data through MCP (Model Context Protocol) servers — custom integrations for Slack, Smartsheet, Google Drive, Outlook, and more. The GSlides MCP server is a custom Node.js server built to extend Claude's reach into Google Workspace.
