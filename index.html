What are Claude Skills?

Skills are specialized AI agents built on the Anthropic Claude Agent SDK. Each skill is a self-contained SKILL.md file that gives Claude a specific identity, toolset, and knowledge base — turning it from a general assistant into a domain expert. Skills are invoked via slash commands (e.g. /pmaas) or natural language triggers and can connect to live tools like Jira, Slack, Smartsheet, GitHub, and Google Drive.

To install: copy a skill's SKILL.md into ~/.claude/skills/<skill-name>/ — or run the one-liner install command on any skill card below. Requires Claude Code.

How It Works

Architecture

Each skill is a SKILL.md file with a YAML frontmatter block defining its name, description, and trigger conditions. When invoked, Claude reads the skill file and adopts the defined persona and toolset. Skills access live data through MCP (Model Context Protocol) servers — custom integrations for Slack, Smartsheet, Google Drive, Outlook, Jira, GitHub, and more.

Install any skill in 30 seconds

Skills live in ~/.claude/skills/. To install PMaaS manually:

mkdir -p ~/.claude/skills/pmaas
curl -sSL https://raw.githubusercontent.com/goyalsandeep2k/claude-skills/main/skills/pmaas/SKILL.md -o ~/.claude/skills/pmaas/SKILL.md

Then open Claude Code and say: "Set up PMaaS for my program"
